\section{Enhance Faith}\label{sec:enhanceFaith}
\textbf{Cost:} 500 LP\\
\textbf{Requirements:} -\\
\textbf{Passive, Repeatable}\\
Whenever you gain a rank in the ~\nameref{perk:increaseFaith} Perk, the amount of points you gain is increased by one die size (from 1d6 to 1d8 to 1d10 to a maximum of 1d12).
Don't re-roll previous ranks of the pool value.
Instead, you gain 1 point for each rank you already have for that pool value.
